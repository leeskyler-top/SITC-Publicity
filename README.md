# 上海信息技术学校 团委学生会 宣传部 管理系统 前端git仓库

主要技术栈：
NodeJS
Vite + Vue3 (Composition API).
Ant-Design-Vue 4.

## 部署与项目更新

### 克隆本项目
```
git clone https://github.com/leeskyler-top/SITC-Publicity.git
```

### 安装npm
```
apt install npm
```

### 安装项目依赖

```sh
npm install
```

### 预览与热重载

```sh
npm run dev
```

### 项目打包

```sh
npm run build
```
